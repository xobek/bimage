## Installation

Install the repo:

```sh
pip install https://github.com/xobek/bimage.git
```


## Usage

To get help with the commandline arguments use:

```sh
bimage --help
```



# Development

Clone the repository and install packages from the requirements.txt file 

```sh
pip install -r requirements.txt
```
